## Descrição
Implemente um programa em Java que:
- Gere uma lista com 100.000 números inteiros aleatórios;
- Implemente dois algoritmos de ordenação (ex.: BubbleSort e QuickSort);
- Meça o tempo de execução de cada algoritmo usando System.nanoTime();
- Compare os resultados em um relatório simples impresso no console, indicando qual algoritmo foi mais rápido e por quê.
 
## Critérios de Avaliação
- 0,25 pt → Geração correta da lista de 100.000 números aleatórios.
- 0,25 pt → Implementação do BubbleSort.
- 0,25 pt → Implementação do QuickSort.
- 0,25 pt → Medição de tempo com System.nanoTime() e relatório comparativo.

## Como Executar

javac Main.java
java Main
